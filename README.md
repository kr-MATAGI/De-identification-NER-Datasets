# De-identification-NER-Datasets

## 비식별 개체명 태그
1) PER
   - 사람 이름
2) ORG
   - 법인명
   - 기업
   - 학교
   - 정부 기관
   - 단체
3) NUM
   - 나이
   - 주민등록번호
   - 신장, 몸무게
   - 우편번호
   - 사업자 등록 번호
   - 카드 번호
   - 비밀 번호
   - 계좌 번호
   - 연락처
   - 재산, 소득
4) LOC
    - 나라
    - 출생지, 거주지
5) INF
   - 일련 번호, 고유 번호
6) DAT
   - 날짜
   - 생년월일
   - 기간
7) HEC (Health Condition)
   - 혈액형, 병, 의료 기록
8) REL (Relation)
   - 가족 관계
9) EDU
    - 자격증
    - 수상 내역
10) POS
     - 직업
     - 직위
